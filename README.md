# Portfolio Project

Welcome to my personal portfolio! This project is a modern, responsive web application built with React and Tailwind CSS, designed to showcase my skills, experience, and projects as a Software Engineer.

## 🚀 Features

- **Modern UI/UX**: Clean, professional design with smooth animations using Framer Motion.
- **Responsive Design**: Optimized for all screen sizes, from mobile to desktop.
- **Interactive Sections**:
  - **Hero**: Eye-catching introduction.
  - **About**: Brief overview of my background.
  - **Projects**: Showcase of my best work with links.
  - **Skills**: Visualization of my technical expertise.
  - **Experience & Education**: Professional and academic timeline.
  - **Contact**: Functional contact form with validation.
- **Glassmorphism**: Aesthetic glass-like components for a premium feel.
- **Typewriter Effect**: Dynamic text in the hero section.

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons, Lucide React
- **Forms**: React Hook Form, Zod, Sonner (for notifications)
- **Email**: EmailJS for contact form integration

## 📦 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/rajkumar-tech-2002/project_portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd project_portfolio/client
   ```
3. Install dependencies:
   ```sh
   npm install
   # or
   pnpm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
